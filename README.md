# WeiboCommentsWebCrawler
用来爬取指定微博博文评论

## 使用方法
更改 01GetWeiboCommets.py 文件中的 id 和 pids 即可。

注意可能还需要改其中 fetchUrl() 函数的 headers 参数。

具体如何获取所需参数，可看我的博客文章[明天写]()。
